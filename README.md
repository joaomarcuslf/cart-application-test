# RailGirls app

## Getting Started
```sh
bundle
rails db:migrate
rails db:seed
rails server
```

## Development Commands

### Show Routes
```sh
rails routes
```

### Show Console
```sh
rails console
```

### Show DB Console
```sh
rails dbconsole
```

# Additional Infos

**Ruby Version**: 2.4.1
**Rails Version**: 5.1.0

## System dependencies

- Rbenv
- Node
